# 🔪 Ciyaal Xamar — Discord Mafia Bot

Discord Mafia ciyaar bot Somali-speaking communities-ka loogu talagalay.

## Sida loo Bilaabiyo

```bash
# 1. Install dependencies
npm install

# 2. .env file samee
cp .env.example .env
# .env ku buuxi DISCORD_BOT_TOKEN-kaaga

# 3. Bot billow
npm start
```

## Amarrada (Commands)

| Amar | Sharaxaad |
|------|-----------|
| `!dilaay` | Lobby cusub bilow |
| `!kasaar` | Host: ciyaaryahan lobby ka saar |

## Doorarka (Roles)

| Door | Sharaxaad |
|------|-----------|
| 🔪 Dilaaye | Habeenta ciyaaryahan dila |
| 🩺 Dhakhtar | Habeenta hal qof badbaadi |
| 🕵️ Danbi-baare | Habeenta hal qof baari |
| 🏠 Shacab | Maalinta Dilaayaha u codeynta |

## Tirada Doorarka

| Ciyaartoyda | Dilaaye | Dhakhtar | Danbi-baare |
|-------------|---------|----------|-------------|
| 5-9 | 1 | 1 | 1 |
| 10-14 | 2 | 1 | 1 |
| 15-20 | 3 | 1 | 2 |

## Sharaxaad

- **Habeenka** (30 sec) — Dilaaye dila, Dhakhtar badbaadiya, Danbi-baare baaraa — DM buttons
- **Maalinta** (60 sec) — Ciyaartoydu codeysan — channel buttons
- **Guusha** — Shacab: dhammaan Dilaayaha saara · Dilaaye: tiradoodu la siman Shacabka

## Bot Invite

Scopes: `bot` + `applications.commands`  
Permissions: Send Messages, Read Messages, Use Slash Commands, Add Reactions
